# Education Trust Website

A modern, responsive education platform designed to provide free study materials, eBooks, practice tests, and exam preparation resources for students.

## 🌟 Features

✅ **9 Complete Pages**
- Home Page - Hero banner with featured materials
- About Page - Mission, vision, and team information
- Study Materials - Organized by categories (Physics, Math, Reasoning, GK, Current Affairs)
- eBook Library - Filterable grid layout with 8+ books
- Exam Updates - Latest notifications and results
- Practice Quiz - Interactive quizzes with instant scoring
- Result Checker - Find exam results using roll number
- Donate Page - Multiple payment options
- Contact Page - Contact form and FAQ

✅ **Responsive Design**
- Mobile-first approach
- Works seamlessly on all devices
- Bootstrap 5 framework

✅ **Interactive Features**
- Practice quizzes with real-time scoring
- Result checking system
- Donation tracking
- Contact forms
- Smooth animations and transitions

✅ **Modern UI/UX**
- Gradient backgrounds
- Smooth scrolling
- Hover effects
- Loading animations
- Professional color scheme
